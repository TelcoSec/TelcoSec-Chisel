📡 Software Defined Radio

Source-compiled SDR drivers and DSP environments sandbox isolated in a Conda environment.

  • telcosec-sdr (Conda)
    Activate with conda activate telcosec-sdr to use compiled drivers.
  • GNU Radio 3.10
    The primary DSP and transceiver flowgraph development system.
  • UHD (USRP) / HackRF / SoapySDR
    Out-of-the-box hardware drivers and transceiver configuration tools compiled from source.
  • gr-gsm Tools
    GSM air interface receiver & analysis. Run grgsm_livemon or grgsm_scanner.
  • Kalibrate RTL
    Frequency offset calibration for RTL-SDR. Run kal -s GSM900.

📱 UE & Baseband Analysis

Specialized platform auditing utilities for Samsung, Huawei, MediaTek, and Qualcomm basebands.

  • FirmWire Emulation
    Baseband emulation/fuzzing for Samsung Shannon and MediaTek. Run via /opt/telcosec/firmwire/venv/bin/firmwire.
  • MobileInsight
    Cellular traffic capturing & analysis directly from diagnostic ports. Folder: /opt/telcosec/mobileinsight-core.
  • QCSuper
    Passive Qualcomm DIAG log capturing to PCAP. Run: python3 /opt/telcosec/qcsuper/qcsuper.py.
  • MTKClient
    BROM bypass, flashing, and partitioning client for MediaTek chipsets.

💳 SIMtrace & Smartcards

Sniffing and analyzing phone-to-SIM card communication interfaces.

  • Osmocom SIMtrace 2
    Sniff ISO 7816 smartcard protocol transactions. Run simtrace2-list or simtrace2-sniff.
  • Osmocom pySim
    SIM/USIM smartcard programming and profile management. Run: pySim-shell.
  • PCSC Daemon
    PC/SC smartcard interface daemon configured for card reader communication.

⛓️ Core RAN & Signaling

Lightweight over-the-air capturing and RAN link emulation.

  • srsRAN
    4G/5G software radio RAN suite simulating UE and eNodeB/gNodeB.
  • Wireshark & tshark
    Network sniffer with comprehensive dissecting capability for SS7, GSM, and LTE/5G signaling.
  • SIPVicious
    VoIP/SIP scanner and penetration auditing toolsuite.
  • sctpscan
    SCTP port scanner for telecom services (Diameter, S1AP, NGAP). Run: sctpscan.
  • SigPloit
    SS7, Diameter, and GTP signaling auditing and exploitation framework. Folder: /opt/telcosec/sigploit.
  • Diafuzzer
    Orange's Diameter protocol fuzzer. Folder: /opt/telcosec/diafuzzer.
  • Scapy (with SS7/Diameter)
    Python packet generation with built-in M3UA, TCAP, MAP, and Diameter encoders.
